- 외부 API에서 데이터를 로드하고 있습니다.
- 사용 가능한 경우 메모리 내 데이터를 사용하려는 경우
- 팝인을 피하기 위해 이미지가 미리 로드될 때까지 탐색을 지연시키려는 경우
